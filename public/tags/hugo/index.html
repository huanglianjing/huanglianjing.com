Hugo的部署与使用

1. 简介 Hugo 是一个使用 Go 编写的静态网站生成器，很适合用来部署个人博客网站。 官网：https://gohugo.io/ 2. 部署 2.1 在mac部署 安装 Hugo： brew install hugo 创建站点： mkdir hugo cd hugo hugo new site huanglianjing.com cd huanglianjing.com 打开 Hugo 的主题列表挑选主题：https://themes.gohugo.io/ 下载并设置主题，将会将主题文件下载到 themes 文件夹内： git init git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke git submodule add https://github.com/CaiJimmy/hugo-theme-stack.git themes/stack git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/papermod git submodule add https://github.com/Vimux/Mainroad.git themes/mainroad echo "theme = 'papermod'" >> hugo.toml 添加文章，将会在 content/posts 下创建文件，然后在创建的 markdown 文件的开头文本后面追加博文内容： hugo new posts/xxx.md # 将已有内容追加到markdown文件中 cat xxx....
